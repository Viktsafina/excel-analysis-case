# Excel Analytics Cases

Учебный аналитический кейс, выполненный в рамках курса по Excel.

## Описание проекта
В работе проведён анализ транзакционных данных розничной сети с фокусом на:
- динамику транзакций по торговым точкам;
- нагрузку на серверы;
- влияние открытия новой торговой точки (каннибализация);
- использование касс самообслуживания;
- анализ времени проведения транзакций и поиск аномалий;
- оценку эффективности программ лояльности;
- построение интерактивного дашборда в Excel.

## Используемые инструменты
- Microsoft Excel
  - сводные таблицы
  - формулы и расчёты
  - диаграммы
  - интерактивные элементы (выпадающие списки, параметры)
- аналитический подход: формирование гипотез и их проверка на данных

## Структура анализа
1. **Анализ торговых точек**
   - динамика транзакций
   - эффект каннибализации после открытия новой точки
2. **Анализ серверов**
   - распределение нагрузки
   - выявление перегрузок
   - моделирование перераспределения торговых точек между серверами
3. **Кассы самообслуживания**
   - доля транзакций
   - влияние на время обработки платежей
4. **Анализ аномалий**
   - выявление всплесков времени транзакций
   - формирование и проверка гипотез
5. **Программы лояльности**
   - оценка потерь от скидок
   - сравнение старой и новой программ
   - расчёт экономического эффекта
6. **Интерактивный дашборд**
   - фильтрация по торговым точкам
   - выбор валют и диапазонов платежей
   - визуализация ключевых метрик

## Результаты и выводы
- Выявлены причины перегрузки серверов и предложено перераспределение нагрузки.
- Подтверждён эффект каннибализации после открытия новой торговой точки.
- Показано влияние касс самообслуживания на длительность транзакций.
- Новая программа лояльности снижает суммарные потери примерно на 12%.
- Разработан интерактивный дашборд для анализа транзакций.
